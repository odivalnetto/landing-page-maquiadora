# Landing Page - Makeup Tedesco

Projeto desenvolvido para divulgação digital de maquiadora profissional, com foco em design moderno, responsividade e facilidade de contato com clientes.

## 📌 Objetivo

Criar uma landing page profissional para fortalecer a presença digital da maquiadora, permitindo que clientes conheçam os serviços oferecidos e entrem em contato de forma rápida através do WhatsApp e Instagram.

---

## 🚀 Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript

---

## 🎨 Funcionalidades

* Layout moderno e responsivo
* Hero section personalizada
* Sessão "Sobre"
* Sessão de serviços
* Integração com WhatsApp
* Links para Instagram
* Navegação suave entre seções
* Design otimizado para dispositivos móveis

---

## 📱 Responsividade

A landing page foi desenvolvida para funcionar corretamente em:

* celulares
* tablets
* computadores
